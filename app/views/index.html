Changes from v1 to v2
  • Removed the necessity for a director/secretary/partner to set up the Defra ID account and enrol onto the service.
  • Removed the step where the regulator is required to approve the user's access to the service.
  • Added a new declaration and privacy policy page before the user enters the service.
  • Removed the 3rd 'certificate of compliance' step from the producer task list.
  • Added the application, payment and status journeys leading off from the producer task list.
  • Added the journey for reporting placed on the market (POM) data.
