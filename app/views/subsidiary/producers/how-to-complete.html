How to complete a CSV file

Enter the required data in the CSV file and use the following column headers:

  1. ‘organisation_id’ - enter the 6 digit long identification number created when the holding company registered with EPR
  2. ‘subsidiary_id’ - enter the self-assigned id used if you have previously submitted data for this subsidiary
  3. ‘organisation_name’ – enter any names exactly as they appear on Companies House
  4. ‘companies house number’ – enter the 8 digit long number
  5. ‘parent_child' – enter Parent or child
  6. 'franchisee_licensee_tenant' - enter Y if the child entity is a franchisee, licensee or tenant
If you are entering data from an organisation details CSV file, you can use the data in column A to populate Column 1, and Columns B, C and E for columns two to four.
