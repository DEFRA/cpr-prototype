Changes from v2 to v3
  • Tweaked the content to make it clear the user should be using their own Defra and GG accounts.
  • Added a route to find out if the organisation has already been registered in Defra Identity and deal with that scenario.
  • Added a details component to help the user understand what makes someone 'authorised'.
  • Added progress saving (save and exit, save and continue) to the longer journeys.
  • Added a question to find out which nation the organisation is registered in.
  • Split the 'producer types' question into multiple pages, one of each type.
  • Renamed the producer type names to use numbers and added examples to the pages.
  • Renamed the payment details PDF to 'invoice'.
  • Other minor content changes.
