How it works

You need to report data that shows an organisation’s packaging activities over a period of time (‘data period’).

For each data period you need to:

  • report organisation details
  • report packaging data
  • pay fees and get packaging recovery notes (PRNs)

You need to start collecting data before the period starts, and pay fees after it has finished. This means you’ll have tasks for more than one data period at the same time.
